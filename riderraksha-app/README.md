# RiderRaksha Frontend

React-based user interface for RiderRaksha — Real-Time Income Protection for Gig Workers.

## 🚀 Features

* User onboarding and registration
* Dynamic policy selection UI
* Real-time premium calculation
* Claims dashboard with filters
* Admin panel (role-based access)
* Payment simulation flow

## 🛠️ Tech Stack

* React 18
* Vite
* Tailwind CSS / Custom Theme
* Fetch API (no axios)

## ▶️ Run Locally

```bash
cd frontend
npm install
npm run dev
```

App runs at: http://localhost:5173

## 🔐 Notes

* Requires backend running on port 5000
* JWT token stored in localStorage
* Role-based UI (admin/user)
